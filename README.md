## 📌 Sobre o projeto

O **InfraCore** é uma ferramenta de automação para suporte de TI no Windows.

Ele reúne comandos úteis de manutenção e diagnóstico em um único script PowerShell, facilitando rotinas de suporte e resolução rápida de problemas.

---

## ⚙️ Funcionalidades

- Limpeza de DNS  
- Renovação de IP  
- Limpeza de arquivos temporários  
- Informações do sistema  
- Diagnóstico básico de rede
